¿Qué es el login en una aplicación?.
El login es el proceso que permite identificar a un usuario antes de
darle acceso a una aplicación o a ciertas funciones.


¿Por qué algunas rutas deben estar protegidas.?
Algunas rutas deben estar protegidas porque contienen información
privada o acciones importantes que no deben quedar abiertas para
cualquiera.


¿Qué tipo de información no debería exponerse públicamente.?
No se debe exponer públicamente contraseñas, documentos de identidad,
teléfonos, direcciones ni claves de acceso.
Tampoco deben mostrarse tokens, claves de API o datos sensibles de la 
comunidad.
